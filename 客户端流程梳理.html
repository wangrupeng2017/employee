客户端方法
基础方法:
1.连接socket服务器::: 
int connect_server(char[20] ip, int port);
参数: @ip:服务端地址  @port:服务端端口
返回值: 已连接的文件描述符(注意判断<0)


首页方法:
1.打印首页菜单:::
void show_home_menu(void);

2.接收菜单输入(注意:输入格式并清除输入缓冲区):::
int get_home_menu_choose(void);
返回值: 选择的菜单序号

3.解析菜单输入
    3.1: 输入错误, 提示信息, Goto首页菜单
    3.2: 选择登录, Goto登录业务(调用登录业务方法)
    3.3: 选择退出, 结束程序

4.登录业务:::
int login_business(void);
返回值: 0:正常退出  !0:错误退出
    4.1: 交互获取登录信息 用户名/密码 格式化登录数据:::
    int get_login_model(struct LoginModel *model);
    参数: @model:请求信息
    返回值: 0:信息获取成功 !0:获取错误或用户取消操作
    4.2: 发送登录请求, 接收结果:::
    int send_login_request(struct LoginModel *model, struct LoginResultModel *out);
    参数: @model:登录信息  @out:登录结果
    返回值: 0:发送登录请求成功 !0:发送登录请求出错
    4.3: 解析结果
        4.2.1: 登录失败显示错误信息, Goto首页菜单(retrun -1)
        4.2.2: 普通用户登录成功, Goto普通员工业务(调用普通员工业务方法)
        4.2.3: 管理员登录成功, Goto管理员业务(调用管理员业务方法)

返回值: 0:正常退出  !0:错误退出

    4.2: 发送信息查询请求, 接收查询结果:::

    参数: @model:查询信息 @out:查询结果
    返回值: 0:发送请求成功 !0:发送请求出错

5.普通员工信息修改业务:::

返回值: 0:正常退出  !0:错误退出

    5.2: 格式化信息修改数据, 发送信息修改请求, 接收处理结果:::
    int send_employee_modify_request(struct EmployeeModifyModel*model);
    参数: @model:用户信息
    返回值: 0:修改成功 !0:修改出错
    5.3: 解析处理结果
        5.3.1: 修改失败显示错误信息, Goto普通用户菜单
        5.3.2: 更新用户本地信息

6.普通员工退出业务:::
    int employee_quit_business(void);
    6.1: 二次确认退出(默认yes):::
